# 강의 계획서

<br>

### 강의 목표

Eleiva를 활용해 파이썬으로 기초적인 데이터 전처리를 하는 방법에 대해 학습합니다. 

<br>

### 개발 로컬 환경 설명

OS : Windows 10 Home 64bit

플랫폼 : Eleiva

개발 언어 : Python 3.8.6

<br>

### 주차별 수업 계획

본 강의는 아래와 같은 순서로 진행됩니다. 

1 ~ 2 주차 : 강의 계획 및 데이터 전처리 들어가기

3 ~ 7 주차 : 정형 데이터 전처리

8 ~ 10 주차 : 이미지 데이터 전처리

11 ~ 13 주차 : 시계열 데이터 전처리

14 ~ 16 주차 : 자연어 데이터 전처리
